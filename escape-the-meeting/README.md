# Escape the Meeting: RPG Edition

**"The only winning move is to manage your stakeholders."**

A narrative-driven office survival RPG where you navigate a corporate maze and battle managers with dialogue choices.

## Features
- **Narrative Battles**: Instead of running away, you must face your colleagues in turn-based dialogue encounters.
- **Sanity System**: Every bad excuse costs you Sanity. If it hits 0, you Burnout (Game Over).
- **Procedural Office**: Every playthrough is a different office layout.
- **Unique Characters**:
    - **Gary (Project Manager)**: Wants to sync.
    - **Karen (HR)**: Wants you to fill out surveys.
    - **Dave (Junior Dev)**: Broke production.

## How to Play
1. Use **Arrow Keys** to move.
2. **Collide with NPCs** to start a conversation.
3. Choose the **Correct Response** to save your Sanity and time.
4. Find the **EXIT** to escape the office.

## Tech Stack
- **Phaser 3**: Game engine.
- **Vite**: Build tool.
- **TypeScript**: Logic.

## Development
```bash
npm install
npm run dev
```
